Replace placeholder filter logic in static/script.js with functional API integration:

1. **Create fetchSchedule(filters) function**:
   ```javascript
   async function fetchSchedule(filters = {}) {
       // Build query string from filters object
       const params = new URLSearchParams();
       if (filters.date) params.append('date', filters.date);
       if (filters.team) params.append('team', filters.team);
       if (filters.status) params.append('status', filters.status);

       const queryString = params.toString();
       const url = queryString ? `/api/schedule?${queryString}` : '/api/schedule';

       try {
           const response = await fetch(url);
           const data = await response.json();

           if (data.success) {
               return data.data;  // Array of game objects
           } else {
               console.error('API error:', data.error);
               return [];
           }
       } catch (error) {
           console.error('Fetch error:', error);
           return [];
       }
   }
   ```

2. **Create renderSchedule(games) function**:
   - Clear existing table rows: `document.getElementById('scheduleBody').innerHTML = ''`
   - For each game, create a `<tr>` with cells for: date, time, weekDay, homeTeam, score/VS, awayTeam, venue, status
   - Append rows to `#scheduleBody`
   - Handle empty results: show "No games found" message in empty state

3. **Create applyFilters() function**:
   ```javascript
   async function applyFilters() {
       const filters = {
           date: document.getElementById('dateFilter').value,
           team: document.getElementById('teamFilter').value,
           status: document.getElementById('statusFilter').value
       };

       const games = await fetchSchedule(filters);
       renderSchedule(games);
   }
   ```

4. **Update event listeners**:
   - Replace console.log() placeholders with `applyFilters()` calls
   - teamFilter.addEventListener('change', applyFilters)
   - dateFilter.addEventListener('change', applyFilters)
   - statusFilter.addEventListener('change', applyFilters)

5. **Update resetFilters handler**:
   ```javascript
   resetFilters.addEventListener('click', async function() {
       document.getElementById('teamFilter').value = '';
       document.getElementById('dateFilter').value = '';
       document.getElementById('statusFilter').value = '';

       // Fetch and display all games
       const games = await fetchSchedule({});
       renderSchedule(games);
   });
   ```

6. **Load initial data on page load**:
   - At end of DOMContentLoaded, call `applyFilters()` to load initial unfiltered schedule

**Implementation notes:**
- Use async/await for fetch calls (cleaner than .then() chains)
- Immutable approach: never mutate DOM state, always rebuild from API response
- Error handling: log errors but show empty table (graceful degradation)
- No console.log statements in production code (per coding-style.md)
