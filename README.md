# CBA赛程查询网站

这是一个CBA（中国男子篮球职业联赛）赛程查询网站，提供赛程加载、筛选和显示功能。

## 功能特点

- **赛程加载**：点击按钮加载CBA第二阶段赛程
- **球队筛选**：按球队筛选赛程
- **日期筛选**：按日期范围筛选赛程
- **场地筛选**：按比赛场地筛选
- **响应式设计**：适配各种屏幕尺寸

## 文件结构

- `cba_schedule.html` - 主页面
- `cba_style.css` - 样式表
- `cba_script.js` - JavaScript逻辑
- `cba_test.html` - 测试页面
- `test_cba.html` - 另一个测试页面

## 使用方法

1. 打开 `cba_schedule.html` 文件
2. 点击"加载赛程"按钮
3. 使用筛选控件查看特定赛程

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- 响应式设计

## 部署

可以直接在GitHub Pages上部署：
1. 进入仓库设置
2. 启用GitHub Pages
3. 选择master分支作为源
