# CBA赛程查询网站

这是一个CBA（中国男子篮球职业联赛）赛程查询网站，提供实时赛程数据查询、筛选和显示功能。

## 功能特点

- **真实数据**：数据来源于新浪体育CBA数据库
- **赛程查询**：显示2025-26赛季完整赛程
- **多维筛选**：按球队、日期、比赛状态筛选
- **实时状态**：显示未开始、进行中、已结束状态
- **比分显示**：已结束比赛显示最终比分
- **响应式设计**：完美适配手机和电脑

## 数据来源

数据来源于 [新浪体育CBA数据库](https://cba.sports.sina.com.cn/cba/)

## 文件结构

- `cba_schedule.html` - 主页面
- `cba_style.css` - 样式表
- `cba_script.js` - JavaScript逻辑
- `vercel.json` - Vercel部署配置

## 使用方法

1. 打开网站
2. 赛程自动加载
3. 使用筛选控件查看特定球队或日期的赛程
4. 点击"刷新数据"获取最新信息

## 技术栈

- HTML5
- CSS3 (响应式设计)
- JavaScript (ES6+)
- 新浪体育CBA数据源

## 部署

已部署到 Vercel，支持GitHub自动部署。
